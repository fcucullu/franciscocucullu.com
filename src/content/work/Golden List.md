<h4>Business Case Context</h4>
<p>Maintaining meaningful relationships requires intentional effort. Golden List was born from the realization that we often lose touch with important people simply because life gets busy. This mobile-first Progressive Web App helps users set customizable reminders for each contact, track interaction history, and build a habit of staying connected with friends, family, and professional contacts.</p>

<p>The app features Google OAuth authentication, push notifications to remind users when it's time to reach out, and a clean dashboard that surfaces who needs attention. Built as a PWA, it works seamlessly on mobile devices with an app-like experience — no app store required.</p>

<h4>Resources Used</h4>

<h5>Frontend</h5>
<ul>
    <li>Next.js 16 – React framework with App Router and server-side rendering</li>
    <li>React 19 – dynamic and responsive user interface</li>
    <li>Tailwind CSS 4 – utility-first styling</li>
    <li>TypeScript – type-safe development</li>
    <li>Lucide React – modern icon library</li>
</ul>

<h5>Backend & Database</h5>
<ul>
    <li>Supabase – authentication (Google OAuth), PostgreSQL database, and real-time APIs</li>
    <li>PostgreSQL – relational database with row-level security policies</li>
    <li>Web Push API – browser push notifications via VAPID keys</li>
</ul>

<h5>Infrastructure</h5>
<ul>
    <li>Vercel – deployment, serverless functions, and cron jobs</li>
    <li>Cloudflare – DNS management and CDN</li>
    <li>GitHub Actions – CI/CD pipeline for automated deployments</li>
    <li>Git – version control</li>
</ul>
